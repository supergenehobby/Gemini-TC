   - 복구: 자동 재시도 (3회), 로컬 캐시 사용
   - 사용자 메시지: "인터넷 연결을 확인하세요"

2. **타임아웃**
   - 원인: 응답 지연
   - 복구: 작업 취소, 상태 롤백

   - 원인: 잘못된 입력
   - 복구: 입력 거부, 가이드 제시
   - 사용자 메시지: "올바른 형식으로 입력하세요"

2. **데이터 손상**
   - 원인: 동기화 실패
   - 복구: 이전 버전 복원
   - 사용자 메시지: "데이터 복구 중..."
