   - 원인: 잘못된 입력
   - 복구: 입력 거부, 가이드 제시
   - 사용자 메시지: "올바른 형식으로 입력하세요"

2. **데이터 손상**
   - 원인: 동기화 실패
   - 복구: 이전 버전 복원
   - 사용자 메시지: "데이터 복구 중..."
